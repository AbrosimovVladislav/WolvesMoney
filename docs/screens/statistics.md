# Экран: Stats (Statistics)

**Компонент:** `src/components/Statistics.tsx`
**Вкладка:** Stats (четвёртая)

---

## Назначение

Визуальная статистика по всем тренировкам и игрокам. Экспорт данных в CSV.

---

## Структура экрана

### 1. Заголовок + кнопки экспорта
- Заголовок "STATS"
- **[↓ Training]** — экспорт CSV тренировок
- **[↓ Players]** — экспорт CSV игроков

### 2. Пустое состояние
Если тренировок нет:
```
📊
No data yet. Add trainings to see stats!
```

### 3. График "Collected vs Ice Cost per Training"
- Столбчатый график, последние 10 тренировок
- Каждая группа: два столбца рядом
  - Зелёный — Collected
  - Красный — Ice Cost
- Высота столбца = `value / maxValue * 100%`
- Подпись оси X: дата в формате `D/M`
- Легенда: ● Collected / ● Ice Cost

### 4. График "Team Balance Over Time"
- Столбчатый график, последние 12 тренировок
- Один столбец на тренировку = накопительный баланс команды на тот момент
- Синий градиент если баланс ≥ 0, красный если < 0
- Высота относительна к диапазону min/max значений

### 5. "Top Contributors (All Time)"
- Топ-8 игроков по сумме всех платежей за всё время
- Каждый игрок:
  - Позиция (1., 2., 3.…) или 🏆 для первого места (золотой цвет)
  - Имя
  - Сумма RSD (зелёный, моно-шрифт)
  - Прогресс-бар (относительно максимума)

### 6. "Training Log" (таблица)
Таблица всех тренировок, от новых к старым:
| Date | Collected | Ice | Result |
|------|-----------|-----|--------|
| D Mon | N RSD | N RSD | ±N RSD |

- Date — синий (`--ice`)
- Collected — зелёный
- Ice — красный
- Result — зелёный если ≥ 0, красный если < 0

---

## Экспорт CSV

### Training CSV (`ice_wolves_finance.csv`)
```
Date,Collected,Ice Cost,Result,Balance
YYYY-MM-DD,N,N,N,N
...
```
`Balance` — накопительный баланс команды после каждой тренировки

### Players CSV (`ice_wolves_players.csv`)
```
Name,Balance,Total Paid
Имя,N,N
...
```

---

## Бизнес-правила

- Графики используют данные из стейта — вычисляются на клиенте
- `balanceData` — накопительный счёт: `runBal += result` для каждой тренировки хронологически
- Top Contributors ограничен 8 игроками
- Bar chart ограничен последними 10 (collected/cost) и 12 (balance) тренировками
