# Экран: Ice Time (Trainings)

**Компонент:** `src/components/Trainings.tsx`
**Вкладка:** Ice Time (третья)

---

## Назначение

Список всех тренировок команды с финансовыми итогами. Отсюда можно создать новую тренировку, удалить существующую или перейти к вводу платежей.

---

## Структура экрана

### 1. Заголовок + кнопка "New"
- Заголовок "TRAININGS"
- Кнопка **[+ New]** — открывает модальное окно создания

### 2. Пустое состояние
Если тренировок нет — отображается:
```
🏒
No trainings yet. Create your first one!
```

### 3. Список тренировок
Карточки отсортированы от новых к старым (по дате).

Каждая карточка содержит:
- **Дата** тренировки (uppercase, `--font-display`)
- **Notes** — заметка, если указана (серый, мелкий шрифт)
- **Тег результата** (+/− RSD, зелёный/красный)
- **Кнопка удаления** [🗑]
- Три мини-блока:
  - **Collected** — собрано RSD (зелёный)
  - **Ice Cost** — стоимость льда RSD (красный)
  - **Paid** — кол-во заплативших из общего (синий, формат `N/M`)
- Подпись внизу: "TAP TO ENTER PAYMENTS →"

Нажатие на карточку (кроме кнопки удаления) → переход на экран **Payments**.

---

## Модальное окно: New Training

Поля:
| Поле | Тип | Дефолт |
|------|-----|--------|
| Date | date picker | сегодня |
| Ice Cost (RSD) | number | 18 000 |
| Notes | text | пусто |

Кнопки: **Cancel** / **Create**

После создания:
- Тренировка создаётся с `total_collected = 0`, `result_balance = -ice_cost`
- Toast: "Training created"

---

## Удаление тренировки

При нажатии 🗑:
- Тренировка удаляется из БД
- Командный баланс корректируется: `teamBalance -= training.result_balance`
- Toast: "Training removed"

---

## Навигация к Payments

Нажатие на карточку тренировки устанавливает `paymentTraining` в родительском компоненте `page.tsx`, что рендерит `PaymentsView` вместо текущего экрана и скрывает нижнюю навигацию.

---

## Бизнес-правила

- Тренировки сортируются по дате убывания (самые новые вверху)
- Статистика (`collected`, `paid`) вычисляется на лету из массива `payments` в стейте
- `Paid` = кол-во уникальных игроков с `amount > 0` для данной тренировки
