# Экран: Dashboard (Home)

**Компонент:** `src/components/Dashboard.tsx`
**Вкладка:** Home (первая, открывается по умолчанию)

---

## Назначение

Главный экран — финансовая сводка команды. Отображает ключевые метрики одним взглядом: баланс команды, долги, кредиты и итоги последней тренировки.

---

## Структура экрана (сверху вниз)

### 1. Hero-блок (Team Bank)
- Логотип/иконка Ice Wolves + название "ICE WOLVES / Finance Dashboard"
- Крупная цифра баланса команды (44px, `--font-display`)
  - Белый цвет если баланс ≥ 0
  - Красный если < 0
- Тег "Team Bank" (зелёный/красный)
- Прогресс-бар заполненности (100% = 50 000 RSD)

### 2. Карточки-метрики (2×2 сетка)
| Карточка | Значение | Цвет |
|----------|---------|------|
| Team Balance | баланс команды в RSD | белый/красный |
| Total Debt | сумма всех отрицательных балансов игроков | красный |
| Credits | сумма всех положительных балансов игроков | зелёный |
| Trainings | количество тренировок всего | белый |

### 3. Блок "Last Training"
Показывается только если есть хотя бы одна тренировка.
- Дата последней тренировки
- Тег результата (+/− RSD)
- Три мини-карточки:
  - **Collected** — сколько собрано (зелёный)
  - **Ice Cost** — стоимость льда (красный)
  - **Result** — разница (зелёный/красный)

### 4. Блок "All-Time Summary"
Строчный список:
- **Total Collected** — всего собрано за всё время
- **Total Ice Costs** — всего потрачено на лёд
- **Net Result** — итоговый результат (собрано − стоимость льда)

### 5. Блок должников (Debtors Alert)
Показывается только если есть игроки с отрицательным балансом.
- Красная предупреждающая рамка
- Заголовок: "N Player(s) Owe Money"
- Список первых 3 должников с суммой долга
- Если должников > 3: "+N more"

---

## Вычисляемые данные

```
totalDebt    = sum(balance < 0 ? -balance : 0) для всех игроков
totalCredit  = sum(balance > 0 ? balance : 0) для всех игроков
lastTraining = тренировка с самой поздней датой
lastResult   = sum(payments where trainingId = lastTraining.id) - lastTraining.iceCost
totalCollectedAllTime = sum(all payments)
totalIceCost = sum(training.iceCost for all trainings)
debtors      = players where balance < 0
```
