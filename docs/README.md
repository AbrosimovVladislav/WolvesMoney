# Ice Wolves Finance — Документация

Приложение для управления финансами хоккейной команды **Ice Wolves**.
Отслеживает оплату тренировок, балансы игроков и общий баланс команды.

## Общая концепция

Приложение работает по следующей модели:

1. В команде есть **игроки** с накопительным балансом
2. Проводятся **тренировки** с фиксированной стоимостью льда
3. После каждой тренировки вводятся **платежи** от каждого игрока
4. Стандартный взнос = **1 500 RSD** за тренировку
5. Если игрок платит больше/меньше стандарта — разница отражается на его личном балансе
6. **Баланс команды** = сумма всех результатов тренировок (собрано − стоимость льда)

## Навигация

Приложение — **мобильное SPA** (max-width 480px).
Нижняя навигационная панель содержит 4 вкладки:

| Иконка | Вкладка | Компонент |
|--------|---------|-----------|
| 🏠 | Home | `Dashboard` |
| 👥 | Players | `Players` |
| 🏒 | Ice Time | `Trainings` |
| 📊 | Stats | `Statistics` |

При нажатии на тренировку в Ice Time — открывается экран **Payments** (нижняя панель скрывается, показывается кнопка «Назад»).

---

## Валюта

Все суммы в **RSD (сербский динар)**.
Форматирование: `1 500 RSD`, сокращённо `1.5k`.
